# NYC Taxi Fare Prediction

## Overview
Machine Learning project to predict NYC taxi fares using trip details.

## Dataset
NYC Taxi Fare Prediction Dataset

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-Learn
- XGBoost
- Matplotlib

## Workflow
1. Data Cleaning
2. Feature Engineering
3. Model Training
4. Hyperparameter Tuning
5. Evaluation using RMSE

## Results
Achieved competitive validation RMSE on the test dataset.
